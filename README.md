## Before we start

#### OSX
get python-3.4.0-macosx10.6.dmg
ctrl-click, open with..., installer (default)

#### Win
get python-3.4.0.amd64.msi

#### Linux
install python-3.4
sudo python3.4 -m ensurepip

#### Libraries
pip3.4 install virtualenv
virtualenv-3.4 akai
source akai/bin/activate
pip install ipython
pip install flake8
